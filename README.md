# Mini Chat App

A simple and efficient public chat application using Node.js, Express, and Socket.IO.

## Features
- Anonymous users with random usernames (e.g., Guest_1234)
- Real-time messaging in a global chat room
- User list display
- Timestamps and usernames for each message
- Clean, modular, and easy to deploy

## Getting Started

1. Install dependencies:
   ```cmd
   npm install
   ```
2. Start the server:
   ```cmd
   npm start
   ```
3. Open your browser and go to `http://localhost:3000`

## Deployment
- Ready for deployment on Render, Vercel, or any Node.js hosting.
